Data Professional Survey Report Dashboard
A fully interactive Power BI dashboard analyzing responses from 216 data professionals across the globe.
This project explores trends in salary, job roles, programming languages, happiness, work–life balance, and the difficulty of breaking into the data field.

📌 Project Overview
This dashboard transforms raw survey data into meaningful insights about the data profession.
It highlights how different roles compare in salary, which programming languages dominate, how satisfied professionals feel, and what challenges newcomers face.

This project demonstrates skills in data cleaning, data modeling, DAX, and dashboard design.

📊 Key Insights
Average Salary by Job Role — Compare compensation across Data Analysts, Engineers, Scientists, Architects, and more.

Happiness & Work–Life Balance — Gauge charts summarizing satisfaction levels.

Demographics — Average age and total survey participants.

Geographic Distribution — Treemap of survey takers by country.

Programming Language Popularity — Usage counts by job title.

Difficulty Entering the Field — Donut chart showing how challenging respondents found it.
